Примеры применения Haskell в реальной жизни

Метки: haskell

Часто спрашивают: “A где-нибудь реально в жизни используют Haskell, кроме написания функции факториала?” Здесь небольшая подборка проектов, в которых используется Haskell. Источником ссылок является неплохая презентация “Intro to Haskell for Erlangers”.

Коммерческое применение Haskell

  • Проект Haxl от Facebook
  • Amgen - применение Haskell в биотехнологиях
  • Количественное моделирование в инвестиционном банке Credit Suisse
  • Создание eDSL для представления и процессинга экзотических вторичных ценных бумаг в Barclays
  • Инфраструктура небольшой торговой группы в Deutsche Bank
  • Платформа для высокочастотных торгов в Tsuru Capital
  • Генерация отчетов в McGraw-Hill Financial с помощью собственного языка отчетов ermine
  • Высокоуровневый язык Bluespec для дизайна микросхем

Приложения написанные на Haskell

  • Silk - платформа для создания коллекций о чем угодно
  • Chordify - создание аккордов по музыкальным композициям для масс
  • Bump (куплен Google в 2013) - обмен файлами, фотографиями, контактами между телефоном и PC (прикрыли проект, архив сайта)
  • MailRank (куплен Facebook, ноябрь 2011) - приоритезация входящих сообщений (прикрыли проект, архив сайта)
  • Bazqux - RSS читалка с комментариями

Коммерческие сервисы

  • janrain - управление профилями клиентов
  • Spaceport (куплен Facebook, август 2013) - фреймворк для мобильных игр с использованием ActionScript 3, компилятор написан на Haskell
  • scrive - сервис для электронной подписи документов (скандинавский рынок)
  • OpenBrain - вычислительная платформа для научной и бизнес-аналитики
  • skedge.me - планирование встреч для бизнеса

Автономные приложения

  • Pandoc - универсальная утилита (“швейцарский нож”) для работы с текстовыми форматами.
  • Darcs - распределённая система управления версиями (типа Git или Mercurial)
  • xmonad - фреймовый оконный менеджер для X Window System
  • Gitit - Wiki с применением Git, Darcs или Mercurial для управления историей изменений
  • git-annex Управление большими файлами с помощью git (похоже на Dropbox)
  • ImplicitCAD - аналог OpenScad, программное моделирование твердотельных трехмерных САПР-объектов
  • Diagrams - DSL для создания векторной графики с использованием Haskell
  • Hakyll - движок для генерации статических сайтов (использован при создании этого блога)
